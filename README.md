# Student_Dashboard_Project 
A modern Student Dashboard web application where students can register, log in, manage their profile, and view their academic information such as enrolled courses and subjects.

 🚀 Features

  🔐 Student Registration & Login
  👤 View Student Profile
  ✏️ Edit Profile Information
  📚 View Enrolled Courses
  📝 View Enrolled Subjects
  🔎 User-Friendly Dashboard Interface
  📱 Responsive Design
